KooDTX - 전문가를 위한 센서 데이터 수집 플랫폼

스마트폰에 내장된 다양한 센서를 활용한 연구나 개발을 하시나요?
데이터 수집, 저장, 분석이 번거로우셨나요?

KooDTX는 모바일 센서 데이터를 쉽고 정확하게 수집할 수 있는 올인원 솔루션입니다.

주요 기능 🎯

✅ 다양한 센서 지원
• 가속도계 (Accelerometer)
• 자이로스코프 (Gyroscope)
• 자기계 (Magnetometer)
• GPS (위치 추적)
• 오디오 녹음 (마이크)
• 기압계, 조도 센서 등

✅ 고정밀 데이터 수집
• 최대 400Hz 샘플링 레이트
• 마이크로초 단위 타임스탬프
• 손실 없는 원본 데이터 저장
• 배치 단위 효율적 저장

✅ 로컬 우선 아키텍처
• WatermelonDB 기반 오프라인 저장
• 네트워크 없이도 작동
• 언제든 데이터 접근 가능
• SQLite 기반 안정적 저장

✅ 스마트 동기화
• Wi-Fi 전용 모드 지원
• 자동 클라우드 백업
• 실패 시 자동 재시도
• 진행 상황 실시간 추적
• 배치 업로드로 효율성 극대화

✅ 강력한 데이터 관리
• 세션별 데이터 그룹화
• CSV/JSON 형식으로 내보내기
• 실시간 차트 시각화
• 오디오 재생 및 공유 기능
• 세션 메모 및 태그

✅ 전문가 수준의 기능
• 샘플링 레이트 커스터마이징
• 다중 센서 동시 녹음
• 데이터 무결성 검증
• 상세한 메타데이터
• 진단 및 성능 모니터링

• React Native 0.73 기반
• WatermelonDB (SQLite) 오프라인 스토리지
• TypeScript로 작성된 안정적 코드
• Material Design 3 UI
• Offline-First Architecture
• RESTful API 동기화

주요 센서 상세 📱

가속도계
- 3축 (X, Y, Z) 가속도 측정
- 최대 400Hz 샘플링
- 중력 및 선형 가속도 구분

자이로스코프
- 3축 각속도 측정
- 회전 감지 및 추적
- 고정밀 움직임 분석

자기계 (Magnetometer)
- 지자기장 측정
- 방향 추적
- 나침반 기능

GPS
- 위도, 경도, 고도
- 속도 및 방향
- 정확도 정보
- 1-10Hz 샘플링

오디오
- 48kHz 고품질 녹음
- 스테레오 지원
- AAC 인코딩
- 음압 레벨 표시

개인정보 보호 🔒

• 모든 데이터는 로컬 우선 저장
• 동기화는 사용자가 완전히 제어
• HTTPS 암호화 전송
• 언제든 데이터 삭제 가능
• 제3자와 데이터 공유 안 함
• GDPR 준수

권한 안내 ⚠️

앱이 요청하는 권한과 사용 목적:

• 위치 (GPS): 위치 데이터 수집
• 마이크: 오디오 녹음
• 저장소: 데이터 파일 저장
• 네트워크: 클라우드 동기화
• 센서: 센서 데이터 접근

모든 권한은 해당 기능 사용 시에만 요청됩니다.

자주 묻는 질문 ❓

Q: 무료인가요?
A: 네, KooDTX는 완전 무료입니다.

Q: 인터넷 없이 사용 가능한가요?
A: 네, 오프라인에서 완벽하게 작동합니다. 동기화만 인터넷이 필요합니다.

Q: 데이터는 얼마나 저장되나요?
A: 디바이스 저장 공간이 허용하는 만큼 저장됩니다.

Q: iOS 버전은 언제 나오나요?
A: iOS 버전은 개발 중이며, 곧 출시 예정입니다.

Q: 상업적 사용이 가능한가요?
A: 네, 연구 및 상업적 목적 모두 사용 가능합니다.

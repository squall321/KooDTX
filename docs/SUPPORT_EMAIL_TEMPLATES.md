## 3. 동기화 문제 (Sync Issues)

### 템플릿 3-1: 동기화 문제 해결 가이드

**본문**:
```
안녕하세요 [이름]님,

KooDTX 팀입니다.
동기화 문제로 불편을 드려 죄송합니다.

동기화가 안 되는 경우 다음 사항을 확인해 주세요:

**1. WiFi 연결 확인**
- WiFi가 켜져 있는지 확인
- 인터넷 연결 테스트 (브라우저에서 웹사이트 열기)

**2. 서버 주소 확인**
- 설정 > 동기화 설정 > 서버 주소 확인
- 연구자에게 제공받은 주소와 일치하는지 확인

**3. API 키 확인**
- 설정 > 동기화 설정 > API 키 확인
- API 키가 만료되었을 수 있으니 연구자에게 문의

**4. 수동 동기화 시도**
- 세션 목록에서 동기화하려는 세션 선택
- "수동 동기화" 버튼 클릭
- 오류 메시지가 나타나면 스크린샷 첨부하여 회신

**5. 연결 테스트**
- 설정 > 동기화 설정 > "연결 테스트" 버튼 클릭
- 결과 확인

위 단계를 시도한 후에도 문제가 지속되면,
다음 정보와 함께 회신해 주세요:
- 오류 메시지 스크린샷
- 로그 파일 (설정 > 고급 > 로그 내보내기)

감사합니다.
KooDTX 지원팀
```

## 4. GPS 정확도 문제 (GPS Accuracy)

### 템플릿 4-1: GPS 정확도 개선 가이드

**본문**:
```
안녕하세요 [이름]님,

KooDTX 팀입니다.

GPS 정확도는 다음과 같은 방법으로 개선할 수 있습니다:

**1. 위치 정확도 개선 (Android 설정)**
- 설정 > 위치 > 위치 정확도 개선
- "고정확도 모드" 활성화
- "WiFi 스캔" 활성화 (WiFi 꺼진 상태에서도 위치 정확도 향상)

**2. 실외 사용**
- GPS는 실내에서 정확도가 낮습니다 (건물, 나무에 가려짐)
- 가능하면 하늘이 열린 실외에서 사용하세요

**3. GPS 워밍업 대기**
- 녹음 시작 후 1-2분 대기 (GPS가 위성과 연결되는 시간)
- 정확도가 20m 이하로 떨어지면 양호한 상태입니다

**4. 기기별 차이**
- GPS 정확도는 기기마다 다릅니다
- 일반적으로 최신 기기가 더 정확합니다

**참고**:
- 앱 내에서 현재 GPS 정확도를 확인할 수 있습니다 (녹음 화면)
- 정확도: 5m 이하 (우수), 5-20m (양호), 20m 이상 (부족)

위 방법을 시도한 후에도 정확도가 낮으면 다시 연락 주세요.

감사합니다.
KooDTX 지원팀
```

## 6. 배터리 소모 문의 (Battery Drain)

### 템플릿 6-1: 배터리 소모 최적화 가이드

**본문**:
```
안녕하세요 [이름]님,

KooDTX 팀입니다.

KooDTX는 센서를 지속적으로 사용하므로 배터리 소모가 발생합니다.
다음 방법으로 배터리 사용을 최적화할 수 있습니다:

**1. 센서 빈도 조정**
- 설정 > 센서 설정 > 샘플링 빈도
- 기본값 50Hz를 10Hz 또는 5Hz로 낮추기
- 정확도는 약간 낮아지지만 배터리 소모 50% 감소

**2. GPS 정확도 조정**
- 설정 > GPS 설정 > 정확도
- "고정확도"를 "균형" 또는 "절전"으로 변경
- 배터리 소모 30% 감소 (정확도 약간 낮아짐)

**3. 사용하지 않는 센서 비활성화**
- 설정 > 센서 설정
- 불필요한 센서(자기계, 기압계 등) 비활성화
- 필요한 센서만 활성화

**4. 보조 배터리 사용**
- 장시간 녹음 시 보조 배터리 사용 권장
- 평균 3-5시간 연속 녹음 가능 (기기에 따라 다름)

**참고**:
- GPS + 다중 센서 사용 시 시간당 15-20% 배터리 소모 (일반적)
- 최적화 후 시간당 8-10% 소모로 감소 가능

감사합니다.
KooDTX 지원팀
```

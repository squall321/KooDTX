# User Support Guide

**사용자 지원 가이드** | User Support Process

이 문서는 KooDTX 앱의 사용자 지원 프로세스를 정리한 관리자용 가이드입니다.

---

## 📬 지원 채널

### 1. 이메일 (support@koodtx.com)

**설정:**
- Gmail 또는 Google Workspace 계정 사용
- 라벨(Labels) 설정:
  - `bug`: 버그 리포트
  - `feature-request`: 기능 요청
  - `question`: 일반 문의
  - `crash`: 크래시 관련
  - `urgent`: 긴급 문의
  - `resolved`: 해결 완료

**자동 회신 설정:**
```
안녕하세요, KooDTX 팀입니다.

문의해 주셔서 감사합니다.
귀하의 메시지를 잘 받았으며, 24-48시간 내에 답변드리겠습니다.

긴급한 문제의 경우, 제목에 [긴급]을 표시해 주시면 우선 처리됩니다.

감사합니다.
KooDTX 지원팀
```

### 2. Google Play 리뷰

**모니터링:**
- Play Console > 평점 및 리뷰
- 알림 설정: 새 리뷰 즉시 알림

**답변 우선순위:**
- ⭐ **1-2점**: 24시간 내 답변 (최우선)
- ⭐⭐⭐ **3점**: 48시간 내 답변
- ⭐⭐⭐⭐⭐ **4-5점**: 감사 표현 (선택사항)

### 3. 앱 내 피드백 (Phase 249)

**설정:**
- 앱 내 "피드백 보내기" 기능
- 자동으로 다음 정보 포함:
  - 앱 버전
  - 기기 모델
  - Android 버전
  - 로그 파일 (선택)

## 🎫 지원 티켓 처리 프로세스

### 1단계: 티켓 분류 (Triage)

**목표 시간: 1시간 이내**

새로운 이메일/리뷰를 받으면 다음과 같이 분류:

| 유형 | 라벨 | 우선순위 | 대응 시간 |
|------|------|---------|---------|
| 크래시 (앱 강제 종료) | `crash`, `urgent` | 🔴 Critical | 2-4시간 |
| 데이터 손실 | `bug`, `urgent` | 🔴 Critical | 2-4시간 |
| 기능 불능 (녹음/동기화 안 됨) | `bug`, `urgent` | 🟡 High | 당일 |
| 일반 버그 (UI 깨짐 등) | `bug` | 🟢 Medium | 2-3일 |
| 기능 요청 | `feature-request` | 🟢 Low | 1주일 |
| 일반 문의 (사용법 등) | `question` | 🟢 Low | 2-3일 |

### 2단계: 초기 대응

**목표: 24시간 이내**

**템플릿 사용:**
- `docs/SUPPORT_EMAIL_TEMPLATES.md` 참고
- 공감 + 해결 방법 제시 + 후속 조치

**예시:**
```
안녕하세요 [이름]님,

KooDTX 팀입니다. 불편을 드려 죄송합니다.

말씀하신 [문제]는 [원인] 때문으로 보입니다.
다음 방법으로 해결해 보시겠습니까?

1. [해결 방법 1]
2. [해결 방법 2]

문제가 해결되면 알려주시고, 계속되면 추가 정보를 제공해 주세요.

감사합니다.
KooDTX 지원팀
```

### 3단계: 문제 해결 또는 에스컬레이션

**해결 가능한 경우:**
- FAQ 링크 제공
- 단계별 가이드 제공
- 스크린샷/비디오 첨부

**에스컬레이션 필요 시:**
- Sentry에서 크래시 로그 확인
- GitHub 이슈 생성
- 개발 팀과 협의

### 4단계: 후속 조치 및 마무리

**해결 확인:**
- 사용자에게 "문제가 해결되었나요?" 확인
- 만족도 조사 (선택사항)

**티켓 종료:**
- `resolved` 라벨 추가
- 필요 시 FAQ에 추가

## 📋 자주 발생하는 문제 및 해결 방법

### 문제 1: 앱 크래시 (강제 종료)

**진단:**
1. Sentry에서 해당 사용자의 크래시 로그 검색
   - 기기 모델, Android 버전으로 필터링
2. 스택 트레이스 확인

**일반적인 원인 및 해결:**
- **메모리 부족**: 다른 앱 종료, 기기 재부팅 권장
- **권한 문제**: 위치 권한 "항상 허용"으로 변경
- **특정 기기 버그**: Hotfix 또는 다음 업데이트에서 수정

**템플릿 답변:**
```
안녕하세요,

크래시 로그를 확인한 결과, [원인]으로 보입니다.
다음 업데이트(v0.X.X, [예정일])에서 수정될 예정입니다.

불편을 드려 죄송하며, 업데이트 후에도 문제가 지속되면 다시 연락 주세요.

감사합니다.
```

### 문제 2: 동기화 실패

**진단 체크리스트:**
- [ ] WiFi 연결 상태 확인
- [ ] 서버 주소 정확성 확인
- [ ] API 키 유효성 확인
- [ ] 서버 상태 확인 (서버 운영자에게 문의)

**해결 방법:**
1. **수동 동기화 시도**
   - 세션 목록 > 세션 선택 > "수동 동기화"

2. **서버 연결 테스트**
   - 설정 > 동기화 설정 > "연결 테스트" 버튼

3. **로그 확인**
   - 설정 > 고급 > 로그 보기
   - 오류 메시지 확인

**템플릿 답변:**
```
안녕하세요,

동기화 문제는 다음과 같은 원인일 수 있습니다:
1. WiFi 연결 문제
2. 서버 주소/API 키 오류
3. 서버 점검 중

[단계별 해결 방법]을 시도해 보시고, 계속 문제가 발생하면 다음 정보를 제공해 주세요:
- 오류 메시지 스크린샷
- 로그 파일 (설정 > 고급 > 로그 내보내기)

감사합니다.
```

### 문제 3: GPS 정확도 낮음

**원인:**
- 실내 사용 (GPS 신호 약함)
- GPS 워밍업 부족 (위성 연결 전)
- 기기 설정 문제

**해결 방법:**
1. **위치 정확도 개선**
   - 설정 > 위치 > 위치 정확도 개선 > 고정확도 모드

2. **WiFi 스캔 활성화**
   - 설정 > 위치 > WiFi 스캔 활성화

3. **실외 사용**
   - 하늘이 열린 실외에서 녹음 시작 후 1-2분 대기

**템플릿 답변:**
```
안녕하세요,

GPS 정확도는 다음과 같은 방법으로 개선할 수 있습니다:
[단계별 가이드]

GPS는 실내에서 정확도가 낮으므로, 가능하면 실외에서 사용하시는 것을 권장합니다.

감사합니다.
```

### 문제 4: 백그라운드 녹음 중단

**원인:**
- 배터리 최적화 (Android 제조사별 다름)
- 메모리 부족

**해결 방법:**

**일반 Android:**
1. 설정 > 앱 > KooDTX > 배터리 > 무제한 사용

**삼성 (Samsung):**
1. 설정 > 디바이스 케어 > 배터리 > 백그라운드 사용 제한 > KooDTX 제외
2. 설정 > 앱 > KooDTX > 배터리 > 제한 없음

**샤오미 (Xiaomi):**
1. 설정 > 앱 > KooDTX > 배터리 절약 > 무제한
2. 보안 > 자동 시작 관리 > KooDTX 허용

**화웨이 (Huawei):**
1. 설정 > 배터리 > 앱 시작 관리 > KooDTX > 수동 관리
2. 모든 권한 허용

**템플릿 답변:**
```
안녕하세요,

백그라운드 녹음 중단은 배터리 최적화 때문일 가능성이 높습니다.

[기기 제조사]의 경우 다음과 같이 설정하세요:
[단계별 가이드]

설정 후에도 문제가 지속되면 다시 연락 주세요.

감사합니다.
```

## 🔄 에스컬레이션 프로세스

### Critical 이슈 (크래시, 데이터 손실)

1. **즉시 확인 (1시간 이내)**
   - Sentry 로그 확인
   - 재현 가능 여부 테스트

2. **GitHub 이슈 생성**
   - 제목: `[Critical] [문제 요약]`
   - 라벨: `bug`, `critical`
   - 담당자 지정

3. **Hotfix 검토**
   - 영향 범위 평가 (몇 % 사용자 영향?)
   - Hotfix 필요 시 `docs/HOTFIX_PROCESS.md` 참고

4. **사용자 알림**
   - 문제 인지 및 조치 중 알림
   - 예상 해결 시간 제시

### High/Medium 이슈

1. **GitHub 이슈 생성**
   - 제목: `[Bug] [문제 요약]`
   - 라벨: `bug`
   - Milestone: 다음 릴리스

2. **사용자 알림**
   - 이슈 등록 완료 알림
   - 다음 업데이트에서 수정 예정 안내

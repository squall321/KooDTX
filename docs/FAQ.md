# FAQ (자주 묻는 질문)

**Frequently Asked Questions** | 자주 묻는 질문

KooDTX 앱 사용자들이 자주 묻는 질문과 답변을 정리한 문서입니다.

---

## 📱 앱 기본 사용

### Q1. KooDTX는 어떤 앱인가요?

**A:** KooDTX는 스마트폰의 다양한 센서(GPS, 가속도계, 자이로스코프 등)를 사용하여 고정밀 데이터를 수집하고 클라우드에 동기화하는 연구용 앱입니다. 주로 다음과 같은 용도로 사용됩니다:

- 차량/보행자 이동 경로 연구
- 모션 패턴 분석
- 환경 센서 데이터 수집
- 학술 연구 및 프로젝트

### Q2. 앱은 무료인가요?

**A:** 네, KooDTX는 완전 무료 앱입니다. 광고도 없으며, 인앱 구매도 없습니다.

### Q3. 어떤 Android 버전을 지원하나요?

**A:** KooDTX는 **Android 10 (API 29) 이상**을 지원합니다. 대부분의 2019년 이후 출시된 Android 기기에서 정상적으로 작동합니다.

### Q4. 앱을 처음 실행하면 무엇을 해야 하나요?

**A:**
1. 앱을 실행하면 위치 및 센서 권한 요청이 나타납니다.
2. **"항상 허용"** 또는 **"앱 사용 중에만 허용"**을 선택하세요.
3. 홈 화면에서 **"녹음 시작"** 버튼을 눌러 첫 번째 세션을 시작하세요.

## 🎙️ 녹음 및 데이터 수집

### Q5. "녹음"이란 무엇인가요? 음성이 녹음되나요?

**A:** 아니요! KooDTX의 "녹음"은 **센서 데이터 수집**을 의미합니다. 음성이나 오디오는 전혀 녹음되지 않습니다. 다음과 같은 센서 데이터만 수집됩니다:

- GPS 위치 (위도, 경도, 고도)
- 가속도계 (X, Y, Z축)
- 자이로스코프 (각속도)
- 자기계 (나침반)
- 기압계 (해발 고도 추정)

### Q6. 녹음 중 화면을 끄거나 다른 앱을 사용할 수 있나요?

**A:** 네, 가능합니다! KooDTX는 백그라운드에서도 센서 데이터를 계속 수집합니다. 다만, 일부 기기에서는 배터리 최적화로 인해 백그라운드 작업이 제한될 수 있습니다.

**백그라운드 녹음이 안 되는 경우:**
1. **설정 > 앱 > KooDTX > 배터리**로 이동
2. **배터리 최적화 제외** 또는 **무제한 사용** 설정
3. (삼성 기기) **설정 > 디바이스 케어 > 배터리 > 백그라운드 사용 제한**에서 KooDTX 제외

### Q7. 녹음 중 앱이 강제 종료되면 데이터가 사라지나요?

**A:** 아니요! KooDTX는 수집된 데이터를 실시간으로 로컬 데이터베이스에 저장합니다. 앱이 강제 종료되더라도 지금까지 수집된 데이터는 안전하게 보존됩니다.

녹음 재개 방법:
1. 앱을 다시 실행
2. **"녹음 재개"** 또는 **"새 세션 시작"** 선택

### Q8. 얼마나 오래 녹음할 수 있나요?

**A:** 이론적으로 무제한이지만, 다음 요인에 따라 제한될 수 있습니다:

- **배터리**: 평균적으로 3-5시간 연속 녹음 가능 (기기에 따라 다름)
- **저장 공간**: 1시간 녹음 시 약 10-20MB 사용 (센서 빈도에 따라 다름)

장시간 녹음 시 보조 배터리 사용을 권장합니다.

## 💾 데이터 저장 및 동기화

### Q9. 수집된 데이터는 어디에 저장되나요?

**A:** 데이터는 두 곳에 저장됩니다:

1. **로컬 저장소** (기기 내부):
   - WatermelonDB를 사용하여 암호화된 SQLite 데이터베이스에 저장
   - 경로: `/data/data/com.koodtx.app/databases/`
   - 다른 앱에서 접근 불가 (Android 보안 샌드박스)

2. **클라우드 (선택사항)**:
   - WiFi 연결 시 자동으로 동기화
   - 서버 주소는 연구자가 제공 (기본값: 없음)

### Q10. 클라우드 동기화는 어떻게 설정하나요?

**A:**
1. **설정 > 동기화 설정**으로 이동
2. **서버 주소** 입력 (예: `https://api.example.com/sync`)
3. **API 키** 입력 (연구자가 제공)
4. **WiFi 전용 동기화** 활성화 (데이터 절약)
5. **자동 동기화** 활성화

동기화는 WiFi 연결 시 자동으로 수행됩니다.

### Q11. 동기화가 안 되는 경우 어떻게 하나요?

**A:** 다음 사항을 확인하세요:

1. **WiFi 연결 확인**
   - WiFi가 켜져 있는지 확인
   - 인터넷 연결 테스트 (브라우저에서 웹사이트 열기)

2. **서버 주소 확인**
   - 설정에 입력한 서버 주소가 정확한지 확인
   - 연구자에게 제공받은 주소와 일치하는지 확인

3. **API 키 확인**
   - API 키가 만료되었거나 잘못 입력되었을 수 있음
   - 연구자에게 문의하여 새로운 API 키 발급

4. **수동 동기화 시도**
   - 세션 목록에서 동기화하려는 세션 선택
   - **"수동 동기화"** 버튼 클릭

문제가 지속되면 support@koodtx.com으로 문의하세요.

### Q12. 데이터를 내보내기(export)할 수 있나요?

**A:** 네, 다음 형식으로 내보낼 수 있습니다:

1. **CSV 형식**
   - 엑셀, Google Sheets에서 열기 가능
   - 센서별로 별도 파일 생성

2. **JSON 형식**
   - 프로그래밍/분석 도구에서 사용 (Python, R 등)

내보내기 방법:
1. **세션 목록**에서 내보낼 세션 선택
2. **"내보내기"** 버튼 클릭
3. 형식 선택 (CSV/JSON)
4. 저장 위치 선택 (Downloads 폴더 권장)

## 🔐 개인정보 및 보안

### Q13. KooDTX는 어떤 데이터를 수집하나요?

**A:** KooDTX는 다음 데이터만 수집합니다:

**수집되는 데이터:**
- GPS 위치 정보 (위도, 경도, 고도, 정확도)
- 센서 데이터 (가속도계, 자이로스코프, 자기계, 기압계)
- 타임스탬프 (데이터 수집 시간)

**수집되지 않는 데이터:**
- ❌ 음성/오디오 녹음
- ❌ 사진/동영상
- ❌ 연락처, 메시지, 통화 기록
- ❌ 설치된 앱 목록
- ❌ 브라우저 기록
- ❌ 개인 식별 정보 (이름, 이메일, 전화번호 등)

자세한 내용은 [개인정보처리방침](../store-assets/legal/privacy_policy.md)을 참고하세요.

### Q14. 내 위치 정보는 안전한가요?

**A:** 네, 안전합니다:

1. **로컬 저장소 암호화**
   - WatermelonDB를 사용하여 데이터베이스 암호화
   - 다른 앱에서 접근 불가

2. **선택적 클라우드 동기화**
   - 사용자가 서버 주소를 입력하지 않으면 동기화되지 않음
   - WiFi 연결 시에만 동기화

3. **사용자 제어**
   - 언제든지 세션 삭제 가능
   - 앱 삭제 시 모든 데이터 자동 삭제

### Q15. 제 데이터는 누가 볼 수 있나요?

**A:**

- **로컬 데이터**: 본인만 볼 수 있습니다 (기기에 저장됨)
- **클라우드 데이터**: 연구자(서버 운영자)만 접근 가능
  - 연구자는 귀하의 위치 정보를 연구 목적으로만 사용할 수 있습니다
  - 제3자에게 판매하거나 공유하지 않습니다

연구 참여에 동의한 경우, 데이터는 익명화되어 학술 연구에 사용될 수 있습니다.

### Q16. 데이터를 삭제할 수 있나요?

**A:** 네, 언제든지 삭제할 수 있습니다:

**로컬 데이터 삭제:**
1. 세션 목록에서 삭제하려는 세션 길게 누르기
2. **"삭제"** 선택
3. 확인

**모든 데이터 삭제:**
1. **설정 > 고급 > 모든 데이터 삭제**
2. 확인 (되돌릴 수 없으니 주의!)

**클라우드 데이터 삭제:**
- 서버에 저장된 데이터 삭제를 원하시면 support@koodtx.com으로 요청하세요.

## 🔧 문제 해결

### Q17. 앱이 자주 강제 종료됩니다.

**A:** 다음 해결 방법을 시도하세요:

1. **앱 업데이트 확인**
   - Google Play 스토어에서 최신 버전으로 업데이트

2. **배터리 최적화 해제**
   - 설정 > 앱 > KooDTX > 배터리 > 무제한 사용

3. **앱 캐시 삭제**
   - 설정 > 앱 > KooDTX > 저장공간 > 캐시 삭제

4. **앱 재설치**
   - 중요: 앱 재설치 전 데이터를 내보내세요 (모든 세션이 삭제됨)

문제가 지속되면 support@koodtx.com으로 크래시 로그와 함께 문의하세요.

### Q18. GPS 정확도가 낮습니다.

**A:** GPS 정확도를 높이는 방법:

1. **위치 모드 변경**
   - 설정 > 위치 > 위치 정확도 개선 > **고정확도 모드** 활성화

2. **WiFi 스캔 활성화**
   - 설정 > 위치 > WiFi 스캔 > 활성화

3. **실외에서 사용**
   - GPS는 실내에서 정확도가 낮습니다 (건물, 나무에 가려짐)
   - 하늘이 열린 실외에서 사용하세요

4. **GPS 워밍업 대기**
   - 녹음 시작 후 1-2분 대기 (GPS가 위성과 연결되는 시간)

### Q19. 앱이 배터리를 많이 소모합니다.

**A:** 배터리 소모는 센서 사용 특성상 불가피하지만, 다음 방법으로 줄일 수 있습니다:

1. **센서 빈도 조정**
   - 설정 > 센서 설정 > 샘플링 빈도 > **5Hz 또는 10Hz**로 낮추기 (기본값: 50Hz)

2. **GPS 정확도 조정**
   - 설정 > GPS 설정 > 정확도 > **균형** 또는 **절전 모드**

3. **사용하지 않는 센서 비활성화**
   - 설정 > 센서 설정 > 불필요한 센서(자기계, 기압계 등) 비활성화

4. **보조 배터리 사용**
   - 장시간 녹음 시 보조 배터리 사용 권장

### Q20. "저장 공간 부족" 오류가 발생합니다.

**A:**

1. **불필요한 세션 삭제**
   - 세션 목록에서 오래된 세션 삭제
   - 또는 내보내기 후 삭제

2. **기기 저장 공간 확보**
   - 다른 앱 삭제 또는 사진/동영상 정리
   - 최소 500MB 이상 여유 공간 권장

3. **데이터 내보내기 및 삭제**
   - 중요한 세션을 CSV/JSON으로 내보낸 후 앱에서 삭제

## 🆘 지원 및 문의

### Q21. 앱에 대해 문의하려면 어떻게 하나요?

**A:** 다음 방법으로 문의하실 수 있습니다:

1. **이메일**: support@koodtx.com
   - 24-48시간 내 답변 (평일 기준)
   - 크래시, 버그, 기능 요청 등

2. **앱 내 피드백**
   - 설정 > 피드백 보내기
   - 자동으로 앱 버전, 기기 정보 포함

3. **Google Play 리뷰**
   - 별점 1-2점 리뷰는 24시간 내 우선 답변

### Q22. 버그를 발견했어요. 어떻게 보고하나요?

**A:** 다음 정보와 함께 support@koodtx.com으로 이메일을 보내주세요:

1. **버그 설명**: 어떤 문제가 발생했나요?
2. **재현 단계**: 어떻게 하면 버그가 다시 발생하나요?
   - 예: "녹음 시작 → 백그라운드로 전환 → 30분 후 앱 열기 → 크래시 발생"
3. **기기 정보**:
   - 기기 모델 (예: Samsung Galaxy S23)
   - Android 버전 (예: Android 13)
   - 앱 버전 (설정 > 정보에서 확인)
4. **스크린샷**: 가능하면 오류 화면 스크린샷 첨부

### Q23. 새로운 기능을 요청할 수 있나요?

**A:** 네, 언제든지 환영합니다!

**기능 요청 방법:**
1. support@koodtx.com으로 이메일
2. 제목: `[기능 요청] 제목`
3. 내용:
   - 원하는 기능 설명
   - 왜 이 기능이 유용한지 설명
   - (선택) 유사한 앱의 예시

모든 요청은 검토 후 우선순위에 따라 개발 계획에 반영됩니다.

## 🌐 기타

### Q24. KooDTX는 오픈소스인가요?

**A:** 현재는 오픈소스가 아닙니다. 추후 공개 여부는 검토 중입니다.

### Q25. iOS 버전은 없나요?

**A:** 현재는 Android 전용입니다. iOS 버전은 사용자 수요에 따라 추후 개발을 고려하고 있습니다.

### Q26. 앱이 데이터 요금을 사용하나요?

**A:** 기본 설정에서는 **WiFi 전용 동기화**가 활성화되어 있어 모바일 데이터를 사용하지 않습니다.

모바일 데이터 사용을 허용하려면:
1. 설정 > 동기화 설정 > **WiFi 전용** 비활성화

주의: 동기화 시 세션당 5-20MB 정도 사용될 수 있습니다.
